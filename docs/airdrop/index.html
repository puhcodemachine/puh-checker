> АИРДРОП · МОНЕТИЗАЦИЯ

Роспись по каждому кошельку: найденная сумма в $ (розданные тебе аирдропы — токены и балансы) и ссылка для управления/продажи. Где пусто — НЕТ.

⚠ Честно: единого API «все аирдропы планеты» не существует (агрегаторы закрыты, бесплатные чекеры — скам). Здесь показана реально найденная ценность по нашим сканам (claimed-аирдропы = токены в кошельках, фильтр скама ≥$1). Для невостребованных (unclaimed) аирдропов — отдельные claim-страницы проектов или платный агрегатор.
